Управление танком №1:
    w - движение вверх
    s - двжение вниз
    a - целиться вверх
    d - целиться вниз
    j - стрелять из арбалета (зажать для увеличения мощности)
    k - стрелять из дробовика (зажать для увеличения мощности)
  
Управление танком №2:
    Up - движение вверх
    Down - двжение вниз
    Left - целиться вверх
    Right - целиться вниз
    z - стрелять из арбалета (зажать для увеличения мощности)
    x - стрелять из дробовика (зажать для увеличения мощности)
    
Цели движущиеся по окружности дают 2 очка

Треугольник в центре "рожает" цели которые движутся прямолинейно и дают 1 очко

Игроки могут соревноваться кто наберет больше очков при стрельбе по целям
